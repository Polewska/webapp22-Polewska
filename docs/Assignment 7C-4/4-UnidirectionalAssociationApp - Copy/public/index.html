Sunny Rehabilitation Resorts

JS/Firebase Web App with Unidirectional Associations

  • Employees
  • Resorts

This app, which is accompanied by a tutorial, is the 4th example app of our incremental sequence of six examples of JS/Firebase Web Apps. This app supports the following operations:

  • Manage resort data
  • Manage employee data

For playing with the app, you may first want to and then view it (e.g. by choosing "Manage employees" and then "Retrieve and list all employees"). For observing the internal status and error messages, turn on the JavaScript console.

This JavaScript front-end data management app takes care of the object types Resort and Employee as well as the two unidirectional associations that assign a manager (Employee) and (one or more) therapists (Employees) to a resort. The app supports the four standard data management operations (Create/Retrieve/Update/Delete). It extends the Validation App by adding code for handling unidirectional associations.
