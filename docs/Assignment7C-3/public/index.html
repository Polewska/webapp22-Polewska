This app, which is accompanied by a tutorial, is the 3rd example app of our incremental sequence of six examples of Plain JavaScript and Firebase Web Apps. This app supports the following operations:

  • Create a new employee record
  • Retrieve and list all employee records
  • Update an employee record
  • Delete an employee record
    •

    • For playing with the app, you may first want to and then view it with Retrieve and list all employees. For observing the internal status and error messages, turn on the JavaScript console.

    This JavaScript front-end data management app takes care of only one object type ("employees") for which it supports the four standard data management operations (Create/Retrieve/Update/Delete). It extends the validation app (described in Part 2 of our tutorial) by adding enumeration attributes, but it needs to be extended by adding further important parts of the app's overall functionality.
